Registro de Mantenimientos

Bienvenidos
A que máquina deseeas registrarle un Mantenimiento.

  • Cortadoras
  • Extruders
  • Mezcladoras
  • Impresoras

Objetivo

Esta página está creada con el fin de llevar un orden de los mantenimientos requeridos para la maquinaria de ROYAL PLASTIC. Podrán llenar un formulario para cada máquina, agregando información específica. Se cuenta con una página para las máquinas cortadoras y otra página para las máquinas extruders y otra para las máquinas impresoras y una más para las mezcladoras.

Instrucciones

  • En "Número de cortadora" o "Extruder", añade únicamente valores numéricos enteros.
  • En "Descripción", puedes añadir letras y números.
  • En "Insumos necesarios", puedes añadir letras y números.
  • En "Suministros necesarios", puedes añadir letras y números.
  • En "Fecha de Registro", lo puedes agregar del calendario proporcionado o manualmente, como mejor lo creas conveniente.
  • En "Fecha de Realización", lo puedes agregar del calendario proporcionado o manualmente, como mejor lo creas conveniente, únicamente cuando ya se haya realizado.
  • En "Costo", añade únicamente valores numéricos enteros y/o valores numéricos decimales.
  • En "Costo extranjero", añade únicamente valores numéricos enteros y/o valores numéricos decimales.
  • En "Tipo de cambio", añade únicamente valores numéricos enteros y/o valores numéricos decimales.
